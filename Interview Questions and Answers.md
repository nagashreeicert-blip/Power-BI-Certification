1. What is Power BI?

Answer:
Power BI is a business analytics tool by Microsoft used to visualize data, create interactive reports, and share insights across an organization.

2. What are the main components of Power BI?

Answer:
Power BI Desktop, Power BI Service, Power BI Mobile, Power BI Gateway, and Power BI Report Builder.

3. What is Power BI Desktop?

Answer:
A Windows application used to import, clean, model data, and create dashboards and reports.

4. What is Power Query?

Answer:
A data transformation tool in Power BI used to load, clean, merge, and shape data.

5. What is DAX?

Answer:
DAX (Data Analysis Expressions) is a formula language used in Power BI to create measures, calculated columns, and custom calculations.

6. What are Visualizations in Power BI?

Answer:
Charts, graphs, maps, cards, KPI indicators, and other elements used to visually represent data.

7. What is a Dataset in Power BI?

Answer:
A collection of data imported or connected to Power BI that is used to build reports.

8. What is a Dashboard?

Answer:
A single-page, interactive canvas in Power BI Service that displays key metrics using tiles.

9. What is a Report?

Answer:
A multi-page visualization document created in Power BI Desktop and published to Power BI Service.

10. What is Power BI Service?

Answer:
An online platform where users publish, share, collaborate, and manage Power BI reports and dashboards.

11. What is Data Modeling in Power BI?

Answer:
The process of creating relationships between tables and designing the data structure for accurate reporting.

12. Difference between Calculated Column and Measure?

Answer:

Calculated Column: Computed during data refresh; stored in the model.

Measure: Calculated at query time; used for aggregations and dynamic calculations.

13. What is the Star Schema?

Answer:
A data modeling approach where a central fact table connects to multiple dimension tables.

14. What is a Relationship in Power BI?

Answer:
A link between two tables based on a common column to enable accurate cross-filtering.

15. What is DirectQuery?

Answer:
A connection mode where Power BI queries data directly from the source without importing it.

16. Difference between Import Mode and DirectQuery?

Answer:

Import Mode: Data is loaded into Power BI; fast performance.

DirectQuery: Data stays in the source; real-time but slower.

17. What is Row-Level Security (RLS)?

Answer:
A method to restrict data visibility for specific users based on roles.

18. What is a Gateway?

Answer:
A bridge that connects on-premises data sources (SQL, Oracle, etc.) to Power BI Service for scheduled refresh.

19. What is Drill-through and Drill-down?

Answer:

Drill-down: View data at detailed levels within a visual.

Drill-through: Open a new page to see filtered details of a specific item.

20. What is a KPI in Power BI?

Answer:
A visual used to track performance against a target value.

21. What is DAX CALCULATE()?

Answer:
CALCULATE modifies the filter context and is used for advanced calculations.

22. Explain Filter Context vs. Row Context.

Answer:

Row Context: Evaluates an expression for each row.

Filter Context: Applies filters before the expression is calculated.

23. What is Composite Model in Power BI?

Answer:
A model that allows combining Import and DirectQuery sources in the same report.

24. What is Incremental Refresh?

Answer:
A feature that refreshes only recent partitions of data instead of the entire dataset â€” improves performance.

25. What is the Use of ALL() in DAX?

Answer:
Removes filters from a table or column for calculations like percent of total.

26. Explain Evaluation Context in DAX.

Answer:
The combination of row context and filter context that determines result outputs.

27. What is Bookmarking in Power BI?

Answer:
A feature to capture report states (filters, visuals) for storytelling or navigation.

28. What is Field Parameter in Power BI?

Answer:
Allows users to dynamically switch measures or dimensions in visuals.

29. What is Paginated Reporting?

Answer:
Pixel-perfect, printable reports created using Power BI Report Builder (similar to SSRS).

30. What is Dataflows?

Answer:
Reusable, cloud-based ETL pipelines in Power BI Service built using Power Query Online.
